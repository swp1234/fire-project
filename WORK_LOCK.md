## 사용법

### 작업 시작할 때
1. 이 파일을 읽는다
2. "현재 진행중" 테이블에 내 작업을 추가한다
3. 작업을 수행한다

### 작업 완료할 때
1. "현재 진행중"에서 해당 행을 삭제한다
2. "완료된 작업"에 옮긴다

### 담당자 표기
- `CC` = Claude Code (CLI)
- `CR` = Cursor

### 주의사항
- **같은 파일을 동시에 수정하지 않는다**
- 진행중인 앱 디렉토리 전체를 잠금 단위로 본다 (예: `projects/idle-clicker/` 전체)
- 포털(`projects/portal/`) 수정은 한 명만
