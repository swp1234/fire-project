## 3. SNS 바이럴 전략

- 게임 플레이 영상 → TikTok/Instagram Reels/YouTube Shorts
- 테스트 결과 공유 → 카카오톡/Twitter
- 결과 카드 이미지 자동 생성 (Canvas → 다운로드)
- Open Graph 메타태그 최적화 (카카오/인스타 미리보기)

## 4. Google Play 출시 가이드

### 준비물
- Google Play Console 개발자 계정 ($25 1회)
- PWABuilder (https://pwabuilder.com) → AAB 생성
- 스토어 자산: 아이콘(512x512), 스크린샷(2~8장), 그래픽(1024x500)
- 스토어 설명 (12개 언어)

### 출시 순서
1. PWABuilder에서 AAB 생성 (앱당 5분)
2. Play Console에서 앱 등록
3. 스토어 자산 업로드 (아이콘/스크린샷/설명)
4. 콘텐츠 등급 설문 작성
5. 출시국 선택 (전체 공개)
6. 심사 제출 → 1~7일 대기

### 출시 우선순위
1. 게임 5개 (수익성 최고)
2. 바이럴 테스트 7개 (유입 채널)
3. 유틸 앱 12개 (보조 수익)

## 5. 백링크 전략

- GitHub 저장소 description에 dopabrain.com 링크
- Dev.to/Hashnode 블로그에 앱 링크 포함
- 무료 디렉토리 등록 (Product Hunt, AlternativeTo)
- 기존 59개 블로그 간 내부 링크 강화
