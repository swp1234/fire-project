## Project Overview

**10-Year Wealth Project: AI 기반 앱/게임/웹 수익 파이프라인**

**Vision:** 10년 내 경제적 자유 달성 - 앱/게임/웹 3채널 광고 수익

**Core Strategy: 적게, 제대로, 빨리**
- 양산보다 **선택과 집중** (수익 가능성 높은 것에 올인)
- **게임**이 유틸 앱보다 광고 수익 5~10배 (체류시간 + 자연스러운 전면광고)
- **바이럴 가능한 앱** (SNS 공유 → 자연 유입)
- **웹 SEO** (Google 검색 유입 → AdSense 수익)

**Roadmap:**
1. **Phase 1 (완료):** 98개 앱/게임 구축 + 웹 포털 + AdSense 승인
2. **Phase 2 (Now → 전환 준비):** 쿠키커터 리디자인 완료 + SEO/Organic 성장 + 바이럴 최적화
3. **Phase 3 (다음):** 캐주얼 게임 고도화 (수익 메인 엔진)
4. **Phase 4:** 성공한 앱/웹 스케일업

**Phase 2→3 전환 기준:**
- Organic 비율 50%+ 안정 유지 (2주 연속)
- 인덱싱 95%+ (현재 97%)
- 블로그 top-10 키워드 5개+
- 위 3개 충족 시 Phase 3 진입 → 게임 고도화에 집중

**리소스 배분 (70/20/10):**
| 비율 | 영역 | 활동 |
|------|------|------|
| **70%** | 게임 고도화 + 기존 앱 개선 | 체류시간↑, 리텐션↑, 광고 최적화 |
| **20%** | SEO/블로그 | 기존 블로그 순위 개선, 내부링크 |
| **10%** | 신규 실험 | 새 앱은 성과 데이터 기반으로만 |

> **핵심:** "양산"이 아니라 "선택과 집중". 새 앱은 기존 앱 데이터가 뒷받침할 때만.

**수익 채널:**

| 채널 | 플랫폼 | 광고 | 장점 |
|------|--------|------|------|
| **앱** | Google Play | AdMob | 앱스토어 검색 유입, 푸시알림 |
| **게임** | Google Play + 웹 | AdMob + AdSense | 높은 체류시간, 반복 플레이 |
| **웹** | GitHub Pages | AdSense | SEO 검색 유입, 즉시 접근, 공유 쉬움 |

**콘텐츠 우선순위:**

| Tier | 유형 | 이유 | 예시 |
|------|------|------|------|
| 1 | 캐주얼 게임 | 체류시간 길고 광고 노출 많음 | 2048, Flappy Bird |
| 2 | 바이럴 테스트 | SNS 공유로 자연 유입 | 심리테스트, 궁합 |

## 웹 수익화 전략

**1. 통합 포털** - 모든 앱/게임을 dopabrain.com에서 제공
**2. SEO 최적화** - 개별 랜딩 페이지 + schema.org
**3. 바이럴 결과 페이지** - 고유 URL + OG 메타태그
**4. 웹 게임 포털** - 게임 간 상호 연결, 리더보드

## 글로벌 전략

**단계적 글로벌화:**
1. 한국어 우선 출시 → 검증
2. 영어 번역 추가
3. 일본어, 중국어 추가 (아시아)
4. 성과 좋은 앱만 추가 언어

**의사결정 기준 (Stop/Go):**

| 지표 | 집중 투자 | 유지 | 정리 대상 |
|------|-----------|------|-----------|
| 월 활성 사용자 | 1,000+ | 100~999 | 100 미만 |
| 평균 체류시간 | 3분+ | 1~3분 | 1분 미만 |
| 재방문율 | 20%+ | 5~20% | 5% 미만 |

**Stop Criteria (신규 앱):**
- 출시 4주 후 MAU < 50 → 추가 투자 중단
- 블로그 8주 후 클릭 0 → 메타 수정 1회 → 4주 후 재평가 → 여전히 0이면 방치
- 게임 체류시간 < 1분 → UX 개선 1회 → 미개선 시 정리 대상
