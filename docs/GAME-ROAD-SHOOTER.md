# Road Shooter - Game Design Document v2

> "1명에서 군대로" — 도로 위 성장형 분대 슈팅 러너
> 영감: Last War: Survival 광고 스타일 + Count Masters + Crowd City
> 목표: **100일 플레이타임** (일 15~30분 세션, 런당 3~5분)
> 플랫폼: HTML5 Canvas 2D (브라우저/모바일), localStorage 저장

---

## 핵심 콘셉트

기존 기획(고정 3~5명 영웅 RPG)을 폐기하고, **런 중 성장**에 집중하는 새로운 게임플레이:

```
[시작: 1명] → 도로 위 아이템 수집 → [10명] → 적 웨이브 격파 → [50명] → 보스전 → [100명+]
```

**핵심 쾌감**: "나 혼자서 시작했는데 어느새 거대한 군대가 되어 있다"
- 숫자가 눈덩이처럼 불어나는 **시각적 쾌감**
- x2 게이트를 통과하는 순간의 **폭발적 성장감**
- 거대 군대가 보스를 녹이는 **압도적 화력감**

---

## 1. 런 구조 (Run Structure)

### 1.1 기본 런 구조

```
타이틀 → [도로 구간 1] → [전투 구간 1] → [도로 구간 2] → [전투 구간 2]
      → ... → [도로 구간 5] → [전투 구간 5] → [보스 전투] → 결과 화면
```

- **1런 = 5개 도로 구간 + 5개 전투 구간 + 1 보스전**
- **런당 소요 시간**: 3~4분 (도로 구간 각 20~25초 + 전투 구간 각 10~15초 + 보스 20~30초)
- 스크롤 방향: 캐릭터가 화면 하단 → 적이 화면 상단에서 등장 (탑다운 뷰)
- **자동 전진**: 캐릭터는 항상 위로 전진, 플레이어는 좌우 이동만 조작

### 1.2 구간별 역할

| 구간 유형 | 길이 (초) | 목적 | 요소 |
|----------|----------|------|------|
| **도로 구간** | 20~25 | 군대 성장 | +1 아이템, x2 게이트, 파워업, 갈림길 선택 |
| **전투 구간** | 10~15 | 군대 시험 | 적 웨이브, 장애물, 군대 손실 발생 |
| **보스 전투** | 20~30 | 최종 시험 | 보스 HP를 군대 화력으로 녹이기 |

### 1.3 런 내 성장 곡선 (기대값)

| 시점 | 예상 병력 수 | 느낌 |
|------|------------|------|
| 시작 | 1명 | "혼자서 어떻게..." |
| 도로 구간 1 완료 | 8~12명 | "소대가 되었다" |
| 전투 구간 1 후 | 5~8명 | "좀 줄었네..." |
| 도로 구간 3 완료 | 30~50명 | "이제 좀 군대다!" |
| 전투 구간 3 후 | 20~35명 | "그래도 많다" |
| 도로 구간 5 완료 | 80~150명 | "대군이다!!" |
| 보스전 돌입 | 60~120명 | "보스를 녹이자" |
| 보스전 후 | 10~80명 (보스 난이도에 따라) | 결과에 따른 보상 |

### 1.4 연속 런 vs 단일 런

- **기본**: 각 런은 독립적 (병력은 매 런 1명에서 시작)
- **스테이지 맵**: 월드맵에서 스테이지 선택 → 런 시작 → 클리어/실패 → 맵으로 복귀
- **런 사이 메타 화면**: 업그레이드, 스킨 교체, 스테이지 선택

## 2. 성장 메카닉 (Growth Mechanics)

> **설계 원칙**: 기존 "+1/+3/+5/+10"처럼 숫자만 다른 아이템을 폐지.
> 모든 아이템은 **고유한 메카닉**을 가지며, 아이템 간 **시너지/상충** 관계가 존재한다.
> 총 47종 (아이템 35 + 게이트 12) — 기존 16종 대비 x3.

### 2.1 병력 모집 아이템 (6종)

| # | 이름 | 시각 | 효과 | 고유한 점 | 상호작용 |
|---|------|------|------|---------|---------|
| 1 | **정찰 토큰 (Scout Token)** | 초록 원 + 눈 아이콘 | +1, 3초간 전방 아이템/함정 미리보기 | 정보 제공, 배신자 자동 감지 | 자석과 조합 = 미리보기 범위 아이템 흡수 |
| 2 | **집결 깃발 (Rally Flag)** | 노란 삼각 깃발 | 현재 병력의 20% 추가 (최소2, 최대15) | 병력 많을수록 효과 상승 | War Drum 중 습득 시 30%로 상승 |
| 3 | **용병 계약 (Mercenary Contract)** | 금테 스크롤 | +5 임시 용병 (전투에서만 활성, 전투 후 소멸) | 게이트 x2 미적용, "진짜 vs 임시" 구분 | 보스전 직전 최적 |
| 4 | **클론 포드 (Clone Pod)** | 시안 캡슐 | +3, 현재 병종 비율대로 복제 | 병종 구성 유지하며 성장 | 전환 게이트 후 습득 = 원하는 병종 집중 |
| 5 | **부활 제단 (Resurrection Altar)** | 보라 십자가 + 영혼 파티클 | 직전 전투 사망자 최대 8명 부활 | 실력에 따라 가치 변동 (무손실 시 효과 0) | Medic Luna 힐과 중첩 |
| 6 | **징병령 (Conscription Order)** | 빨간 서류봉투 | +8, 단 공격력 50%, 이동 -20% (무기 업글 1회로 해제) | "양 vs 질" 판단 | War Drum으로도 패널티 해제 가능 |

### 2.2 게이트 시스템 (12종)

기존 "수학 연산만"에서 **수량/병종/진형/시간/리스크** 다차원 선택으로 확장.

#### 수량 게이트 (3종, 모든 스테이지)

| 게이트 | 좌측 | 우측 | 핵심 판단 |
|--------|------|------|---------|
| **배수 게이트 (Classic)** | 병력 x2 | 병력 +10 | 10명 기준 손익분기 |
| **도박 게이트 (Gambler's)** | x3 또는 ÷2 (50/50 룰렛) | +5 확정 | 리스크 성향 테스트 |
| **카운트다운 게이트** | 3초 내 통과 = x2, 늦으면 +3 | +8 확정 | 조작 스킬 + 즉시 판단 |

#### 병종/진형 게이트 (3종, 스테이지 5+)

| 게이트 | 좌측 | 우측 | 핵심 판단 |
|--------|------|------|---------|
| **병종 전환 (Class Shift)** | 소총병 → 탱커 | 소총병 → 저격수 | 다음 전투 미리보기 참고 |
| **정예 단조 (Elite Forge)** | 병력 -30%, 생존자 공격력 x2 (영구) | 병력 +20% 변동 없음 | **소수 정예 vs 물량 작전** |
| **진형 게이트 (Formation)** | 쐐기형 고정 (전방 +50%, 후방 피격 -50%) | 원형 (360도 사격, 측면 강함) | 적 등장 패턴에 따라 |

#### 시간/조건 게이트 (3종, 스테이지 10+)

| 게이트 | 좌측 | 우측 | 핵심 판단 |
|--------|------|------|---------|
| **시한폭탄 (Ticking Bomb)** | +15, 15초 후 자폭 (적에게 범위 데미지) | +5 영구 | 전투 시작 타이밍 계산 |
| **기후 게이트 (Weather)** | "추위": 적 이동 -50%, 아군 공속 -30% | "열풍": 아군 공속 +50%, 함정 데미지 x2 | 방어적 vs 공격적 |
| **빚 게이트 (Debt)** | 지금 +20, 다음 게이트에서 -10 차감 | 지금 +5 부채 없음 | 미래 게이트와의 연동 판단 |

#### 고급 게이트 (3종, 스테이지 20+)

| 게이트 | 좌측 | 우측 | 핵심 판단 |
|--------|------|------|---------|
| **제물 게이트 (Sacrifice)** | 탱커 전멸 → 수 x3 소총병 | 저격수 전멸 → 수 x2 폭파병 | 병종 비율이 가치 결정 |
| **거울 게이트 (Mirror)** | 분대 복제 (x2), 8초 후 소멸 | 분대 유지 + 8초 무적 | 보스전 직전 = 좌측(순간 DPS x2) |
| **미스터리 게이트 (Mystery)** | "???" (x3/+20/전환/÷3 랜덤) | +7 확정 | 기대값 +12 vs 최악 ÷3 |

### 2.3 환경 상호작용 아이템 (5종)

도로 환경 자체를 물리적으로 변형시킨다.

| # | 이름 | 효과 | 전략 |
|---|------|------|------|
| 1 | **도로 확장기 (Road Widener)** | 5초 도로 폭 1.5배 → 수집 범위↑, 피탄면적↑ | 아이템 밀집 구간에서 극강, 전투 중 위험 |
| 2 | **연막탄 (Smoke Screen)** | 5초 적 사격 50% miss, 아군 사거리 -40% | 러셔 방어에 탁월, 보스전에서는 DPS↓ |
| 3 | **중력장 (Gravity Well)** | 2초간 주변 적+아이템+함정 전부 흡수 | 아이템 모으기에 좋지만 함정/적도 끌려옴 |
| 4 | **가교 건설기 (Bridge Builder)** | 다음 갈림길 양쪽 모두 통과 가능 | 갈림길 직전에 습득해야 유효 (타이밍 핵심) |
| 5 | **지진 북 (Earthquake Drum)** | 모든 함정 위치 노출 + 독가스 범위 50%↓ | 함정 밀집 구간(월드 7)에서 극대화 |

### 2.4 조건부 아이템 (4종)

특정 조건 만족 시에만 발동. **줍기 전에 조건 확인 필수.**

| # | 이름 | 조건 | 효과 |
|---|------|------|------|
| 1 | **약자의 각성 (Underdog Boost)** | 병력 10명 이하일 때만 | 전군 공격력 x3 + 이동속도 x2 (10초) |
| 2 | **시너지 수정 (Synergy Crystal)** | 3종 이상 병종 보유 시 | 전 병종 공격력 +30% (8초). 2종 이하면 +10%만 |
| 3 | **베테란 훈장 (Veteran's Medal)** | 가장 오래 생존한 병사 1명에게 | HP x5, 공격력 x3, 크기 1.5배 (영웅화) |
| 4 | **보름달 유물 (Full Moon Relic)** | 도로 후반부(70%+)에서만 발동 | 다음 전투 첫 웨이브 적 이동 -80% |

### 2.5 시너지/조합 아이템 (5종)

단독으로도 유효하지만, 조합 시 폭발적 시너지.

| # | 이름 | 단독 효과 | 조합 시너지 |
|---|------|---------|----------|
| 1 | **군가의 북 (War Drum)** | 5초 이동 +30%, 수집 +20% | Rally Flag와 = 모집률 30%. 징병령과 = 패널티 해제 |
| 2 | **탄약 상자 (Ammo Crate)** | 5초 전군 관통 (1체, 뒤 50%) | 저격수 30%+ = 3체 관통, 감쇠 없음 |
| 3 | **봉화대 (Beacon Tower)** | 설치형, 주변 적 초당 10 데미지 | 같은 런 2개 설치 = 화염벽 생성 |
| 4 | **자기 코어 (Magnetic Core)** | 3초 아이템 흡수 + 함정 밀어냄 | 중력장과 조합 = 좋은 것만 흡수, 나쁜 것 배척 |
| 5 | **조준 레이저 (Targeting Laser)** | 5초 전군 단일 대상 집중 (DPS +40%) | 보스전 극대화, Shadow 크리와 조합 |

### 2.6 리스크/리워드 아이템 (5종)

강력하지만 반드시 대가가 따른다. "이걸 줍는 게 맞나?" 고민이 핵심.

| # | 이름 | 보상 | 대가 | 최적 상황 |
|---|------|------|------|---------|
| 1 | **광전사 룬 (Berserker Rune)** | 8초 공격력 x3, 공속 x2 | 8초 후 3초 기절 (무방비) | 전투 시작 직후 (8초 내 웨이브 클리어) |
| 2 | **영혼 수확 (Soul Harvest)** | 적 처치 시 병력 +1 (최대 10) | 습득 시 현재 병력 10% 즉사 | 병력 적을 때 (대가 적고 회수 가능) |
| 3 | **시간 왜곡 (Time Warp)** | 도로 구간 리셋 (아이템 리스폰) | 시간 추가 소비 + 함정도 리스폰 | 아이템 풍부한 구간 한정 |
| 4 | **기생 군단 (Parasitic Swarm)** | +12, 적 처치 시 분열 +1 (최대 20) | 10초마다 일반 병사 1명 감염 (통제 불능화) | 빠르게 전투 끝내는 러시 플레이 |
| 5 | **유리 대포 (Glass Cannon)** | 전군 공격력 x4 (**런 영구**) | 전군 HP 1 고정 (한 방에 사망) | Hell 전용, 패턴 완벽 회피 가능한 고수 |

### 2.7 시간 기반 아이템 (4종)

습득 후 시간에 따라 효과가 변한다. "언제 줍느냐"가 "뭘 줍느냐"만큼 중요.

| # | 이름 | 시간 변화 | 최적 타이밍 |
|---|------|---------|----------|
| 1 | **아드레날린 (Adrenaline Surge)** | 공격력 +100% → 매초 -10% → 10초 후 0% | 전투 시작 직전 1~2초 |
| 2 | **진화 씨앗 (Evolving Seed)** | 즉시 +1 → 5초 +2 → 10초 +3 → 15초 +5 (누적 +11) | 도로 시작 직후 (성장 시간 확보) |
| 3 | **부식 수류탄 (Decay Grenade)** | 화면 내 적 전원 DoT (초당 5, 10초) | 전투 전 적 미리 등장 시 |
| 4 | **모래시계 (Hourglass)** | 6초 적만 슬로우 -60% → 2초 아군 슬로우 -60% | 웨이브 초반 (6초 안에 처치) |

### 2.8 함정 (6종)

각각 고유한 위협 메카닉. 단순 "-N"이 아닌 **질적으로 다른 위험**.

| # | 이름 | 효과 | 고유한 점 |
|---|------|------|---------|
| 1 | **유사 (Quicksand)** | 이동 -90% (3초), 직접 손실 없음 | 후속 위험과의 콤보가 진짜 위협 |
| 2 | **배신자 (Traitor)** | +1로 위장, 3초 후 배신 (주변 3명 피해) | Scout Token이 자동 감지. 관찰력 테스트 |
| 3 | **EMP 지뢰 (EMP Mine)** | 5초간 자동 사격 비활성 (이동은 가능) | 전투 직전에 밟으면 치명적 |
| 4 | **탈영 깃발 (Desertion Flag)** | 가장 약한 5명 이탈 (1명씩 달아남) | 징집병 먼저 탈영, 약한 병사 정리 활용 가능 |
| 5 | **미믹 상자 (Mimic Chest)** | 60% 골드+3 / 40% 폭발(-5+경직) | 기대값 양수이지만 도박 |
| 6 | **중력 역전 (Gravity Flip)** | 3초간 진형 역전 (탱커↔저격수 위치) | 전투 직전 밟으면 저격수가 전방 노출 |

### 2.9 핵심 시너지 매트릭스 (10선)

| 조합 이름 | 아이템 A + B | 결과 | 발견 난이도 |
|----------|------------|------|-----------|
| 물량 폭발 | Rally Flag + War Drum | 모집률 30% | 쉬움 |
| 정예 육성 | Conscription + War Drum | 징집병 즉시 정규병화 | 보통 |
| 완전 흡수 | Road Widener + Magnetic Core | 아이템만 흡수, 함정 배척 | 보통 |
| 저격 관통 | Ammo Crate + 저격수 30%+ | 3체 관통, 감쇠 0% | 어려움 |
| 봉화 연격 | Beacon Tower x2 | 화염벽 생성 | 어려움 |
| 시간 정지 | Full Moon + Weather Gate 추위 | 적 이동 사실상 0 | 매우 어려움 |
| 중력 콤보 | Gravity Well + Magnetic Core | 좋은 것만 흡수 | 어려움 |
| 광전사 부식 | Berserker Rune + Decay Grenade | x3 + DoT = 웨이브 초토화 | 보통 |
| 역전의 영웅 | Underdog + Soul Harvest | 소수 고공격→처치→회복→대군 | 어려움 |
| 씨앗 정찰 | Evolving Seed + Scout Token | 성장 대기 중 전방 정보 확인 | 쉬움 |

### 2.10 아이템 배치 철학

```
[안전 경로: Scout Token, Evolving Seed, Rally Flag]        ← 점진적 성장 + 정보
[위험 경로: Quicksand, Traitor, 끝에 Glass Cannon]          ← 극한 리스크/리워드
[게이트: Elite Forge Gate (정예 vs 물량)]                    ← 근본적 전략 분기
[함정 구간: EMP Mine 사이에 Ammo Crate들]                   ← 조작 스킬 + 보상
[조합 구간: War Drum → Rally Flag → Clone Pod]              ← 시너지 발견의 쾌감
[시간 구간: Evolving Seed(도로 초반) + Adrenaline(전투 직전)] ← 타이밍 계산
```

**5대 원칙**:
1. "안전하지만 느린 길" vs "위험하지만 보상 큰 길" 항상 제공
2. 아이템 조합으로 "유레카" 경험 제공
3. 조건부 아이템으로 매번 다른 최적해
4. "언제 줍느냐"가 "뭘 줍느냐"만큼 중요
5. 리스크 아이템으로 고수/초보 차별화 (초보도 우회 가능)

### 2.11 병력 상한

- **소프트 캡**: 200명 (화면에 표시 가능한 실질 한계)
- **하드 캡**: 999명 (숫자만 표시, 시각적으로는 200명처럼 보임)
- 200명 이상이면 개별 캐릭터 대신 "밀집 군단" 표현 (성능 최적화)

## 3. 캐릭터 타입 (Character Types)

### 3.1 기본 설계: 단일 타입에서 시작 → 점진적 분화

**런 시작 시 1명 = 기본 소총병(Rifleman)**

병력을 얻을 때 기본적으로 소총병이 추가되지만, **특수 모집 아이템**을 통해 다른 타입 획득 가능.

### 3.2 병종 4가지

| 병종 | 시각 | 역할 | 공격 | 사거리 | 체력 | 해금 |
|------|------|------|------|--------|------|------|
| **소총병 (Rifleman)** | 초록 원 + 총 막대 | 기본 DPS | 보통 | 중거리 | 1 | 기본 |
| **탱커 (Tanker)** | 파란 큰 원 + 방패 사각형 | 전방 방어 | 낮음 | 근접 | 3 | 스테이지 5+ |
| **저격수 (Sniper)** | 보라 삼각형 + 긴 막대 | 고데미지 | 높음 | 장거리 | 1 | 스테이지 10+ |
| **폭파병 (Bomber)** | 주황 원 + 원형 폭탄 | 범위 공격 | 매우 높음 (범위) | 중거리 | 1 | 스테이지 15+ |

### 3.3 도로 위 병종 모집 아이템

| 아이템 | 시각 | 효과 |
|--------|------|------|
| **+1 탱커** | 파란 토큰 + 방패 | 탱커 1명 추가 |
| **+2 저격수** | 보라 토큰 + 조준경 | 저격수 2명 추가 |
| **+1 폭파병** | 주황 토큰 + 폭탄 | 폭파병 1명 추가 |
| **병종 전환 게이트** | 아치 문 (해당 병종 색) | 통과한 전체 병력이 해당 병종으로 변환 |

### 3.4 병종 전환 게이트 (전략적 선택)

```
[소총병 → 탱커 게이트]  vs  [소총병 → 저격수 게이트]
```

- 다음 전투가 **다수 약한 적**: 폭파병 변환이 유리
- 다음 전투가 **소수 강한 적**: 저격수 변환이 유리
- 다음 전투가 **보스**: 탱커 + DPS 혼합이 유리
- 게이트 전에 **다음 전투 미리보기** 표시 → 전략적 선택 유도

### 3.5 병종 비율 밸런스 (이상적)

| 상황 | 이상적 비율 | 이유 |
|------|-----------|------|
| 일반 전투 | 소총병 70% + 탱커 15% + 나머지 15% | 범용 화력 |
| 보스전 | 탱커 30% + 저격수 40% + 소총병 30% | 생존 + 단일 화력 |
| 웨이브 전투 | 폭파병 40% + 소총병 40% + 탱커 20% | 범위 화력 |

## 4. 포메이션 시스템 (Formation)

### 4.1 병력 수에 따른 자동 대형

캐릭터들은 병력 수에 따라 **자동으로 대형을 변경**한다.

| 병력 수 | 대형 | 시각적 표현 | 도로 차지 폭 |
|--------|------|-----------|------------|
| 1명 | 단독 | 중앙 한 점 | 10% |
| 2~3명 | 일렬 | 가로 한 줄 | 15% |
| 4~6명 | 삼각형 | 앞 1~2 + 뒤 2~3 | 20% |
| 7~12명 | 다이아몬드 | 다이아몬드 형태 | 30% |
| 13~25명 | 직사각형 | 촘촘한 격자 | 40% |
| 26~50명 | 원형 | 원 형태 밀집 | 55% |
| 51~100명 | 쐐기형 | V자 진형 | 70% |
| 101~200명 | 대형 화살표 | ▶ 형태, 화면 대부분 차지 | 85% |
| 200명+ | 밀집 군단 | 빽빽한 원형, 숫자 카운터 표시 | 90% |

### 4.2 대형 내 병종 배치 규칙

```
          [저격수 저격수]          ← 후방 (뒤쪽 줄)
       [소총병 소총병 소총병]       ← 중간
    [소총병 폭파병 소총병 폭파병]    ← 중간
  [탱커 탱커 탱커 탱커 탱커 탱커]    ← 전방 (앞쪽 줄)
```

- **탱커**: 항상 최전방 배치 → 적 사격을 먼저 맞음
- **소총병**: 중간열 배치 → 기본 DPS 출력
- **폭파병**: 중간열 랜덤 배치 → 범위 공격
- **저격수**: 최후방 배치 → 높은 단일 데미지

### 4.3 대형 시각 효과

- **이동 시**: 캐릭터들이 살짝 출렁이며 이동 (젤리 물리)
- **x2 게이트 통과**: 캐릭터가 "뻥" 하고 분열되는 애니메이션
- **사격 시**: 각 캐릭터에서 총알 파티클 발사
- **사망 시**: 빨간 플래시 + 사라짐 (0.3초 fadeout)
- **대형 변경 시**: 0.5초에 걸쳐 부드럽게 재배치

### 4.4 Canvas 렌더링 최적화

```javascript
// Character representation (simple geometric shapes)
function drawCharacter(ctx, type, x, y, scale) {
  switch(type) {
    case 'rifleman':

    case 'tanker':
      // Blue larger circle + blue rectangle (shield)

      ctx.beginPath();
      ctx.arc(x, y, 6 * scale, 0, Math.PI * 2);
      ctx.fill();
      ctx.fillStyle = '#60a5fa';
      ctx.fillRect(x - 4, y - 6, 8, 3);
      break;
    // ... sniper, bomber similarly
  }
}
```

## 6. 손실 메카닉 (Loss Mechanics)

### 6.1 병력 손실 원인

| 원인 | 손실 규모 | 빈도 | 회피 가능성 |
|------|---------|------|-----------|
| **적 러셔 돌진** | 1~3명 | 높음 | 좌우 회피 가능 |

| **독가스 장판** | 초당 1명 | 보통 | 빠르게 통과 |

### 6.2 사망 처리

- 캐릭터 사망 시: 빨간 플래시 → 0.2초 내 사라짐
- **탱커 우선 피격**: 전방 탱커가 먼저 사격/돌진 흡수
- **최소 보장**: 병력이 1명까지 줄어도 즉시 게임오버 아님 (1명이 0이 되면 게임오버)
- **부활 아이템**: 치유 하트 아이템으로 최근 사망 5명 부활

### 6.3 게임오버 조건

```
병력 0명 = 게임오버
```

- 게임오버 화면에서:
  - **무료 부활**: 광고 시청 → 병력 10명으로 부활 (런당 1회)
  - **재시작**: 해당 스테이지 처음부터 다시
  - **맵으로 돌아가기**: 보상 없이 맵으로 복귀

## 10. 메타 진행 (Meta-Progression)

### 10.1 런 간 영구 업그레이드

런에서 획득한 **골드**와 **스타코인**으로 영구 업그레이드 구매.

#### (A) 기본 스탯 업그레이드 (골드)

| 업그레이드 | 효과 | 최대 레벨 | 비용 (레벨당) |
|-----------|------|---------|-------------|
| **시작 병력** | 런 시작 시 +1명 (기본 1 → 최대 5) | Lv 4 | 500 / 1,500 / 5,000 / 15,000 |
| **기본 공격력** | 전 병종 공격력 +10% | Lv 10 | 200 → 10,000 (지수 증가) |
| **기본 체력** | 전 병종 체력 +1 | Lv 5 | 300 → 8,000 |
| **이동 속도** | 분대 좌우 이동 +5% | Lv 5 | 200 → 3,000 |
| **아이템 자석 범위** | 아이템 흡수 범위 +10% | Lv 5 | 150 → 2,000 |
| **골드 획득량** | 런 중 골드 +10% | Lv 10 | 300 → 12,000 |
| **파워업 지속시간** | 버프 지속시간 +10% | Lv 5 | 400 → 5,000 |

#### (B) 병종 업그레이드 (골드)

| 업그레이드 | 효과 | 최대 레벨 | 비용 (레벨당) |

| **보스 약점** | 보스 약점 표시 (데미지 +25%) | 40 스타코인 |

### 11.1 Core Loop (5초 반복)

```
아이템 수집 (좌우 이동) → 병력 증가 (숫자 UP) → 시각 피드백 (군대 커짐) → 더 모으고 싶음
    ↑                                                                    ↓
    ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

### 11.2 도파민 트리거 매핑

| 트리거 | 발생 빈도 | 도파민 강도 | 설명 |
|--------|---------|----------|------|
| **+1 수집** | 매 1~2초 | ★☆☆☆☆ | 작은 만족, 높은 빈도 = 기본 리듬 |
| **+5, +10 수집** | 매 10~15초 | ★★☆☆☆ | "오 좋다" 중간 보상 |
| **x2 게이트 통과** | 매 30~40초 | ★★★★☆ | "우와!!" 폭발적 성장, 시각 연출 |
| **적 웨이브 섬멸** | 매 40~60초 | ★★★☆☆ | 압도적 화력감, 적 쓸어버리기 |
| **보스 격파** | 런당 1회 | ★★★★★ | 최고 쾌감, 큰 보상, 연출 |
| **200명 달성** | 드묾 | ★★★★★ | 화면 가득 군대, "나 이만큼 모았다" |
| **3-스타 클리어** | 스킬에 따라 | ★★★☆☆ | 완벽 플레이 성취감 |

### 11.3 "한 판만 더" 요소

1. **짧은 런 길이** (3~4분) → "한 판만 더 할 수 있어"
2. **성장의 불확실성** → "이번에는 x2를 더 잘 활용해볼까"
3. **아깝게 실패** → "조금만 더 잘하면 클리어인데..."
4. **업그레이드 직전** → "500골드만 더 모으면 업그레이드 가능"
5. **보스 HP 간당간당** → "이번엔 보스를 잡을 수 있을 것 같아"

### 11.4 이탈 방지 전략

| 이탈 위험 | 해결책 |
|----------|-------|
| "너무 쉬워서 지루" | 스테이지 10부터 갈림길 복잡도 증가, Hard/Hell 난이도 |

### 26.1 오브젝트 풀링

```javascript
class EntityPool {
  constructor(factory, size = 50) {
    this.pool = Array.from({length: size}, () => factory());
    this.active = [];
  }
  acquire() {
    const e = this.pool.pop() || this.factory();
    e.active = true;
    this.active.push(e);
    return e;
  }
  release(e) {
    e.active = false; e.reset();
    this.active.splice(this.active.indexOf(e), 1);
    this.pool.push(e);
  }
}
// Pools: Bullet(100), Enemy(30), Item(20), Particle(200), Character(120)
```

## 27. 오리지널리티 — "이 게임만의 것"

> Road Shooter를 Count Masters/Last War 클론이 아닌, **독자적 게임**으로 만드는 차별화 포인트.

### 27.1 핵심 차별화 5가지

| # | 차별화 | 경쟁작 | Road Shooter |
|---|--------|-------|-------------|
| 1 | **시너지 기반 도로** | 아이템은 단순 +N/xN | 47종 아이템이 상호작용. 조합 발견이 "유레카" 경험 |
| 2 | **무기 = 군대 표현** | 전원 동일 사격 | 18종 무기, 3슬롯 편성. 200명이 각자 다른 무기 = 독특한 시각 |
| 3 | **적이 도로를 바꾼다** | 적은 전투 구간에만 존재 | 도둑/부패자/매설병이 도로 아이템/게이트와 상호작용 |
| 4 | **보스 = 퍼즐** | 보스는 HP 스펀지 | 3~4 페이즈 멀티 메카닉, 관찰+이해 필요 |
| 5 | **리스크 아이템 선택** | 아이템은 항상 이로움 | 광전사 룬, 유리 대포, 기생 군단 등 대가 있는 파워 |

### 27.2 "이 게임에서만 가능한" 경험 6가지

1. **200명 전쟁고 드럼 + 번개 연결 + 산탄포** = 화면 전체가 전쟁 영화
2. **유리 대포(x4, HP 1)로 Hell 보스 노데스 클리어** = 극한 스킬 증명
3. **아이템 도둑을 일부러 먹게 두고 사살 = 2배 보상** = 도둑을 이용하는 전략
4. **게이트 부패자 채널링 중 사살 = x3 강화** = 위기가 기회로
5. **어둠 군주 심연 포탈에 병사 희생 = 대량 역류 데미지** = 숨겨진 메카닉 발견
6. **최종 사령관 P4에서 눈 방향으로 안전칸 판별** = 관찰력 보스전

### 27.3 타겟 플레이어 페르소나

| 페르소나 | 플레이 스타일 | 매력 포인트 |
