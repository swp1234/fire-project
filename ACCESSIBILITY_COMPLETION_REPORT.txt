IMPLEMENTATION DETAILS
================================================================================

Skip Link (HTML)
  <a href="#main-content" class="skip-link">Skip to main</a>

Skip Link (CSS)
  .skip-link {
    position: absolute;
    top: -40px;
    left: 0;
    background: #9b59b6;
    z-index: 100;
  }
  .skip-link:focus { top: 0; }

Focus Visible (CSS)
  button:focus-visible,
  a:focus-visible,
  input:focus-visible,
  [role="button"]:focus-visible {
    outline: 3px solid currentColor;
    outline-offset: 2px;
  }

Motion Support (CSS)
  @media (prefers-reduced-motion: reduce) {
    * {
      animation-duration: 0.01ms !important;
      animation-iteration-count: 1 !important;
      transition-duration: 0.01ms !important;
      scroll-behavior: auto !important;
    }
  }

Touch Targets (CSS)
  button, a[href], input[type="button"],
  [role="button"], [role="link"] {
    min-width: 44px;
    min-height: 44px;
    display: inline-flex;
    align-items: center;
    justify-content: center;
  }

High Contrast (CSS)
  @media (prefers-contrast: more) {
    button, input, a[href], [role="button"] {
      border: 2px solid currentColor;
      font-weight: bold;
    }
  }

APPENDIX: WCAG 2.1 CRITERIA REFERENCE
================================================================================

WCAG Levels:
  A (Minimum): Basic accessibility
  AA (Intermediate): Enhanced accessibility, legally required
  AAA (Advanced): Maximum accessibility optimization

Color Contrast Ratios:
  AA: 4.5:1 for normal text, 3:1 for large text
  AAA: 7:1 for normal text, 4.5:1 for large text

Touch Target Size:
  AA: 44x44px minimum (WCAG 2.1)
  AAA: 48x48px (Apple/Google recommendation)

Focus Indicators:
  Minimum: 2px outline with 2:1 contrast
  Best: 3px outline with high contrast

Motion:
  Users can disable animations: prefers-reduced-motion: reduce
  No auto-playing content > 5 seconds
  No rapid flashing (>3 per second)
